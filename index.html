Rules for naming JS variables

Variable names can only contain letters, digits, underscores (_), and dollar signs ($).
For example: $my_1stVariable

Variables must begin with a letter, underscore (_), or dollar sign ($).
For example: $name, _name, or name

Variable names are case-sensitive.
For example: myVariable and MyVariable are different.

Variable names should not be JS reserved words.
For example, you cannot use var, function, or return as variable names.
